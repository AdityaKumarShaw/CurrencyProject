# 💱 Currency Converter (React)

A simple and responsive **Currency Converter Web App** built using **React + Vite + Tailwind CSS**.
It allows users to convert currencies in real-time using exchange rate data from an API.

---

## 🚀 Live Demo

👉 https://currency-project.vercel.app

---

## 📌 Features

* Convert between different currencies
* Real-time exchange rate fetching
* Swap currencies instantly
* Responsive UI
* Built using modern React practices (Hooks & Components)

---

## 🛠️ Tech Stack

* **React**
* **Vite**
* **JavaScript**
* **Tailwind CSS**
* **Currency Exchange API**

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/AdityaKumarShaw/CurrencyProject.git
```

Navigate to the project folder:

```
cd CurrencyProject
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

---

## 📦 Build for Production

```
npm run build
```
